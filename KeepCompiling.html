Warning: Improper sequence will cause immediate kernel panic and system destruction.

// SECTION 1: POWER INITIALIZATION

If the system timer is currently displaying an EVEN number of seconds when you begin, toggle Switch A to the ON position. Otherwise, leave it OFF.

// SECTION 2: NETWORK BYPASS

Switch B must only be activated AFTER Switch D has been toggled, but BEFORE Switch C.

// SECTION 3: MEMORY FLUSH

Switch C is highly volatile. Toggle it ON only if Switch A is currently OFF. If Switch A is ON, Switch C must remain OFF.

// SECTION 4: CLOCK SYNC

Always activate Switch D first, unless the instruction in Section 1 explicitly states otherwise.

Translate carefully. "Even" = Gerade, "Odd" = Ungerade. "Before" = Vorher, "After" = Nachher.
